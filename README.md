# Employee Management System

A full-stack Employee Management System built with modern technologies and production-oriented architecture.

## Tech Stack

### Backend

* NestJS
* TypeScript

### Frontend

* React
* TypeScript
* Tailwind CSS

### Database

* PostgreSQL
* Prisma ORM

### Authentication

* JWT

## Project Structure

```text
employee-management-system/
├── backend/
├── frontend/
├── docs/
└── docker/
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run start:dev
```

### Frontend *(Coming Soon)*

```bash
cd frontend
npm install
npm run dev
```
